Built For Daily Work

Terminal UX that respects real projects.

Multi-model by default

Use LM Studio local models and DeepSeek today. Provider/model profiles keep names clear: DeepSeek/deepseek-v4-pro.

Fast model switching

/model opens a keyboard picker, aggregates discovered models, and remembers the last selected profile.

Secure remote keys

Remote API keys are stored in macOS Keychain. Config files keep only references, not plaintext secrets.

Built-in MCP tools

Shell, file read/write, structured edits, grep, and web search are ready after ma init.

Slash commands

Type / for suggestions, use arrow keys to choose, and press Tab to complete.

Local instructions and skills

MA reads AGENT.md and project-local .ma/skills/*.md so teams can encode workflow knowledge.

中文

MA 是一个本地优先、多模型的终端编程 Agent。

当前 alpha 支持 LM Studio 本地模型和 DeepSeek 官方 API，内置模型切换、Keychain 密钥存储、MCP 工具、斜杠指令和项目技能。

配置体验

ma init 通过交互式流程选择 provider、发现模型、保存 profile。

模型切换

/model 直接上下选择所有模型，并记住上一次使用的模型。

安全边界

远程 API Key 不写入明文配置，删除和查看走系统认证。
